## Add new SVG Icon

1. put new svg files into `src/uikit/icons/raw`
2. run npm script `yarn gen:icon`

## Usage

```ts
import { Icon } from '@tidb-cloud-uikit/icons'

// with ts typing support!
<Icon name="SomeName" />
<Icon name="SomeName" size={20} color="red" />
```
